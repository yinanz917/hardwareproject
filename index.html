Computers

What is a computer?

A computer is an electronic device for storing and processing data, typically in binary form, according to instructions given to it in a variable program. It has the ability to store, retrieve and process large amounts of data quickly.

How does a computer work?

A computer takes input from the user and outputs information after processing the information and can store it in its memory. It then displays the information using a peripheral such as a monitor. The most simple language of code is called machine code, and it breaks the computer’s tasks into binary, made up of 1s and 0s.

How do the components interact?

Each component of a computer has a major part in the process of displaying the desired information or data to an output device. First, an action is completed by an input device (which are typically peripherals, such as a mouse opening a program), then a copy of the software program is transferred from the hard drive (permanent computer storage) onto the RAM (internal device memory). This is because the current software required by the CPU for processing has to be in temporary storage, the random access memory (RAM). The CPU then performs the tasks in the given order with the help of the clock. The clock gives a pulse beat to the CPU to keep instructions in order and determines the speed at which they are carried out. This prevents the system from crashing.

Parts of a Computer

The System Unit

A plastic or metal box that contains most of the components of the computer (excluding external peripherals).

Motherboard

It acts like the “mother” of the computer as it connects different components. The mother is a printed circuit board (PCB) that contains the main components of a computer. It holds the CPU, memory chips (RAM and ROM), power connectors, and a large number of expansion slots for processors such as the video card and solid state drives.

CPU (Central Processing Unit)

The Central Processing Unit (a.k.a. the microprocessor) is one of the most important components in a computer. This part of the computer is responsible for completing arithmetic and logical operations and processing information from memory. It is the brain of the computer, as it can add, subtract, and compare with binary codes. Binary can be represented easily as “on” and “off” in the CPU by millions of switches called transistors. These transistors carry out the functions of the CPU in binary code.

Hard Drive

The hard drive is vital for the computer to function. It is where all your data is stored permanently, unlike RAM, which is temporary. It has a hard disk inside of it where all your files are physically located. The data is stored on a stack of disks magnetically. These disks spin extremely fast so that the data can be accessed immediately. The hard drive can also be divided into directories (folders).

CD-ROM Drive

The Compact Disk Read Only Memory, is a pre-pressed compact disk which stores data. Computers are able to read the stored data, however, they aren’t able to be changed, only read. The information is usually programmed into the computer during construction. For example, the CD-ROM Drive would contain the data used when the computer is turned on.

Floppy Disk Drive

This component is a drive that is used for floppy disks to transfer information between computers and it also saves information on these disks. Floppy disks are inserted into the drive entrance in order for the computer to use them. Nowadays, floppy disks have become irrelevant after the invention of smaller and more efficient forms of storage, such as a USB flash drive.

RAM (Random Access Memory)

It is a type of memory that the operating system, applications, and data in current use are stored so that it can be accessed much faster from the CPU. RAM is faster to read and write than other forms of storage such as Hard Drives and SSDs. Data is stored in RAM as long as the computer is running, once the computer is shut down, all the data stored in the RAM is lost. It is a form of temporary storage. If data held on the RAM is to be permanently stored, it should be saved to the hard drive.

Sound Card

The sound card is a peripheral that is slotted on the motherboard that allows the computer the use of audio. That provides the desired audio to be outputted to a device.

Graphics Card (Integrated)

The graphics card located on the motherboard is a component that converts data into graphics that are displayed on the monitor. A GPU(graphics processing unit) that is integrated into the motherboard and shares memory with the CPU. Integrated graphics are usually more economical and use less power but have far less processing power than a stand-alone GPU.

Cooling Fan

It is a fan that is connected on top of the heatsink to help cool the heatsink. This component is key to maintaining the functionality of the CPU. The CPU carries millions of electrical signals every second which generates a lot of heat. The cooling fan prevents the CPU from overheating.

Heatsink

The heatsink is a device that is attached to the CPU to cool it down by absorbing heat and dissipating it into the fan which blows it out of the case.

Power Supply

The power supply converts high voltage AC (alternating current) from a socket to a low voltage DC (direct current) to power the computer. All components of the computer are only able to run using DC power. The voltage size it converts to depends on the component and the amount of power it requires.
